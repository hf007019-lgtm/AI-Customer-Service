# 🤖 AI Customer Service System

## 📌 项目简介

一个基于 Python 的 AI 客服系统，支持商品查询和自动回复。

## 🚀 功能

* 用户聊天接口
* AI 自动回复
* 商品数据库查询
* 简单 Web 聊天界面

## 🛠 技术栈

* Python (Flask)
* SQLite
* HTML + JS

## ▶️ 运行方式

```bash
pip install -r requirements.txt
python app.py
```

打开浏览器：

```
frontend/index.html
```

## 💡 示例

用户输入：

> 有什么手机？

系统回复：

> 我们有 iPhone 15，价格是 6999 元
